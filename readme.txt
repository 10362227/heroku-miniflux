go to 'main' branch to deploy

deploy with button
	https://heroku.com/deploy

deploy with cli
	$ bash ./heroku-cli.sh

enable auto update
	1. deploy app with button or cli
	1. fork this repo on github
	2. go to heroku web, then choose your app
	3. set 'deployment method' to github
	3. select your repo, then 'main' branch
	4. click 'enable automatic deploy'
